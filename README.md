## Description:

Simplified analog of slack chat.

## Setup:

1) Clone the repository:

```
git clone git@github.com:DmitriyChestnov/frontend-project-12.git
```

2) Install depencies:

```sh
make install
```

3) Local application:

```sh
make start
```
